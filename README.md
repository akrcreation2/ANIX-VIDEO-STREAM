# Auto Play Using URL

Simply add

```
?video=
```

Example

```
https://akrcreation2.github.io/ANIX-VIDEO-STREAM/?video=https://example.com/movie.mp4
```

Google Drive Example

```
https://akrcreation2.github.io/ANIX-VIDEO-STREAM/?video=https://gdrive-dd.bypased.workers.dev/direct.aspx?id=YOUR_FILE_ID
```

Direct MP4

```
https://akrcreation2.github.io/ANIX-VIDEO-STREAM/?video=https://example.com/video.mp4
```

M3U8

```
https://akrcreation2.github.io/ANIX-VIDEO-STREAM/?video=https://example.com/master.m3u8
```
